# Flight Price Trend & Demand Analysis using Web Scraping

### Project Overview
This project focuses on analyzing airline ticket pricing trends and travel demand patterns by scraping real-time flight data from the EaseMyTrip website. The project combines Web Scraping, Data Cleaning, Exploratory Data Analysis (EDA), and Data Visualization to extract meaningful business insights from airline fare data.

The objective of this project is to understand:
+ Flight pricing behavior
+ Airline-wise fare comparison
+ Travel demand trends
+ Route-based price variations
+ Duration and timing impact on ticket prices
  
### Project Objectives
+ Scrape real-time flight data from EaseMyTrip
+ Analyze airline pricing trends across multiple routes
+ Identify cheapest and expensive travel periods
+ Compare ticket prices among airlines
+ Discover travel demand patterns using visual analytics
+ Generate business insights using EDA techniques

### Dataset Features

The scraped dataset contains:

+ Airline Name
+ Source City
+ Destination City
+ Departure Time
+ Arrival Time
+ Flight Duration
+ Ticket Price
+ Stops Information
  
### Data Preprocessing

Performed several preprocessing techniques to improve data quality:

+ Removed duplicate records
+ Handled missing values
+ Cleaned inconsistent text formats
+ Converted prices and duration into numerical format
+ Standardized airline and route names
+ Performed feature engineering on flight timings
+ Detected and treated outliers
  
### Exploratory Data Analysis (EDA)

Conducted detailed EDA using statistical analysis and visualizations:
+ Univariate Analysis
+ Bivariate Analysis
+ Multivariate Analysis
+ Correlation Analysis
+ Airline-wise Price Comparison
+ Route-wise Demand Analysis
+ Duration vs Ticket Price Analysis
+ Time-based Fare Trend Analysis
### Key Insights
+ Non-stop flights generally have higher ticket prices
+ Morning and evening flights show higher average fares
+ Metro city routes have higher travel demand
+ Ticket prices vary significantly across airlines
+ Travel duration impacts fare pricing trends
